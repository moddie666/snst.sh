# Simple Network Speed Test

### Test network connection speeds with simple command line tools
depends on "nc" and "pv"

```
USAGE: snst.sh (-r|-s <host|ip>) [-p <port>]
-r|--recv   ... listener (must be started first)
-s|--send   ... sender mode
-p|--port   ... port to use
-h|--help   ... this text
```
